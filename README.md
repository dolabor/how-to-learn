## Описание проекта
Данный проект создан с целью закрепления на практике учебного материала по расширенным возможностям HTML и СSS.

Использовались:
* Флексбокс вертска
* iframe
* Анимации и трансформации
* Позиционирование
* Организация файловой структуры по БЭМ (nested)

## Технологии
* HTML5
* CSS3
